
Grid of quadrilaterals with perlin noise made with [purescript-p5](https://github.com/parenparen/purescript-p5).

Inspired by https://www.kovach.me/posts/2018-03-07-generating-art.html

## Installing

```
bower install
npm install
```

## Running

```
npm run webpack:dll
npm run webpack:server
```
